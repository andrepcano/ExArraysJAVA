# 📊 ArraysJAVA

Este repositório foi criado com o objetivo de praticar e consolidar conceitos fundamentais de **arrays em Java**, por meio de diversos exercícios que exploram desde o básico até aplicações mais estruturadas.

---

## 🚀 O que foi aprendido

Durante o desenvolvimento deste projeto, foram aplicados na prática os seguintes conceitos:

### 🔢 Arrays (Vetores)

* Declaração e inicialização de arrays
* Acesso a elementos por índice
* Percorrer arrays com `for` e `for-each`

### 🔁 Estruturas de Repetição

* Uso de `for`
* Iteração eficiente sobre coleções de dados

### 📐 Operações com Arrays

* Soma e média de valores
* Maior e menor elemento
* Contagem de ocorrências

### 🔄 Manipulação de Dados

* Ordenação de arrays
* Busca de elementos (linear)
* Cópia e transformação de arrays

### 🧠 Lógica de Programação

* Resolução de problemas com base em arrays
* Organização de dados em memória
* Pensamento algorítmico

---

## 💻 Tecnologias utilizadas

* Java

---

## 📌 Objetivo do projeto

Este repositório tem como foco o desenvolvimento da lógica de programação utilizando arrays, que são uma das estruturas mais importantes para qualquer programador.

## 📚 Aprendizados importantes

* Como trabalhar com coleções de dados de forma eficiente
* Como resolver problemas utilizando estruturas simples
* Base sólida para avançar para listas (`ArrayList`) e estruturas mais complexas
